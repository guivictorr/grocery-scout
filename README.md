## üì±üíª Instructions

```
## 1. Clone repo
git clone https://github.com/guivictorr/grocery-scout.git

## 2. Change to project folder
cd grocery-scout

## 3. Install dependencies
npm install

## 4. Make sure docker is running

## 5. Run the app
npm run dev
```

You can use Expo Go to scan the QR code and run the app.
